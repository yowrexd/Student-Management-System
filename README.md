# Student Management System

A full-stack student management system with both backend and frontend components.

## Technical Requirements

### Backend
- Built with any backend framework (Laravel, Node.js, Django, etc.)
- Implements RESTful API endpoints
- MySQL database integration
- Features:
  - Student profile management
  - Subject/Course management
  - Grade tracking system
  - Full CRUD operations for all entities
  - Proper error handling and validation

### Frontend
- Built with any frontend framework (React, Vue, Angular) or HTML/CSS/JS
- User interface for:
  1. View and manage student profiles
  2. Display list of subjects per student
  3. Show detailed grade breakdown (activities, quizzes, exams)
  4. Add/edit grades
- Implements basic UI/UX practices for navigation and data input

## System Requirements
- RESTfulAPI Integration
- Proper error handling and validation
- Version control (Git)
- Hosted on Render

## Getting Started (instructions to be updated)

1. Clone the repository

   - Install required dependencies
   - Configure database settings
   - Run migrations
   - Start the server

3. Set up Frontend:
   - Install required dependencies
   - Configure API endpoints
